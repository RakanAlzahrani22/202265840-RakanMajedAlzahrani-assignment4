# AI Usage Report

## AI Tool Used
- ChatGPT

## How AI Was Used

I used ChatGPT as a support tool during the development of this assignment. It helped me improve my website step by step while still allowing me to understand and test each feature myself.

The AI support was mainly used for:
- suggesting interactive features that fit the assignment requirements
- helping debug JavaScript and CSS issues
- improving the contact form validation
- helping organize and write the project documentation

## Specific Ways AI Helped

### 1. Planning interactive features
I used ChatGPT to suggest features that could improve my Assignment 1 portfolio and make it suitable for Assignment 2. Based on these suggestions, I chose features that were practical and matched my project:
- live project search/filter
- contact form validation with feedback
- theme toggle with saved preference using localStorage

### 2. Debugging and fixing errors
I used ChatGPT to help identify and fix problems during development. For example:
- the project search feature did not work at first
- the theme toggle did not visually change the page at first
- a misplaced closing brace in the CSS affected layout and styling

The AI suggestions helped me narrow down the issues, but I still had to test the changes in the browser and confirm which fixes actually worked.

### 3. Improving form validation
In Assignment 1, the feedback showed that the form validation message was weak. For Assignment 2, I used ChatGPT to improve it by:
- adding inline error messages under each field
- validating empty fields
- checking that the email format is valid
- showing a success message after valid submission

This made the form more user-friendly and better aligned with the rubric.

### 4. Writing documentation
I used ChatGPT to help structure and draft:
- README.md
- technical-documentation.md
- ai-usage-report.md

I reviewed and edited the content so it matches my actual project and reflects the work I completed.

## How I Reviewed AI Output

I did not directly submit unmodified AI output. I reviewed suggestions and tested them before keeping them.

My review process included:
- checking whether the code matched my existing project structure
- testing the behavior in the browser
- changing or rejecting suggestions that did not fit my setup
- applying edits gradually instead of replacing the entire project at once

For example, some early suggestions assumed a different file structure, but my existing structure was already working, so I kept the parts that matched my project and ignored the unnecessary parts.

## What I Learned

Using AI in this assignment helped me better understand:
- how JavaScript can update content dynamically based on user input
- how to validate a form and display helpful feedback
- how CSS structure can affect multiple parts of a page
- how localStorage can be used to save user preferences
- how to improve user experience with transitions and interactive behavior

I also learned that AI-generated suggestions still need careful testing, because even correct-looking code can fail if it does not match the actual project.

## Modifications I Made

I modified AI suggestions in multiple ways:
- I applied changes step by step instead of replacing the whole project
- I adjusted code to match my own HTML, CSS, and JavaScript files
- I simplified some ideas to keep the project understandable
- I tested each feature before moving to the next one

## Reflection

I used AI as a learning and support tool, not as a replacement for my own work. The final project reflects my own testing, understanding, and edits.

AI was especially useful for:
- debugging
- suggesting improvements
- explaining problems clearly
- helping me write complete documentation

I understand the final implementation and can explain how the project search, form validation, and theme toggle features work.
